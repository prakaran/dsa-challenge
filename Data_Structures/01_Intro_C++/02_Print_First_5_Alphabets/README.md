# B. Print First 5 Alphabets

| Attribute | Specification |
| :--- | :--- |
| **Time Limit per Test** | 1 second |
| **Memory Limit per Test** | 256 megabytes |

## Problem Description
Your task is to print the first five letters of the English alphabet.

## Input
There is no input for this problem.

## Output
Print the first five uppercase English letters, one per line.

## Examples

### Example 1
**Input:**
```text
There is no input.
```

**Output:**
```text
A
B
C
D
E
```

---

## My Solution Notes
* **Time Complexity:** $\mathcal{O}(1)$ — Printing a fixed number of lines takes constant time.
* **Space Complexity:** $\mathcal{O}(1)$ — No extra memory or dynamic data structures are used.
