# C. Triangle

| Attribute | Specification |
| :--- | :--- |
| **Time Limit per Test** | 1 second |
| **Memory Limit per Test** | 256 megabytes |

## Problem Description
Your task is to print a right-angled triangle pattern using asterisks (`*`).

## Input
There is no input for this problem.

## Output
Print the following pattern exactly as shown:
```text
*****
****
***
**
*
```

## Examples

### Example 1
**Input:**
```text
There is no input.
```

**Output:**
```text
*****
****
***
**
*
```

---

## My Solution Notes
* **Time Complexity:** $\mathcal{O}(1)$ — The pattern has a fixed size and prints a constant number of characters.
* **Space Complexity:** $\mathcal{O}(1)$ — No dynamic memory allocation or variables scale with input.
