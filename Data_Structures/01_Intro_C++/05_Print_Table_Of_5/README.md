# E. Print Table of 5

| Attribute | Specification |
| :--- | :--- |
| **Time Limit per Test** | 1 second |
| **Memory Limit per Test** | 256 megabytes |

## Problem Description
Your task is to print the multiplication table of 5 from 1 to 10.

## Input
There is no input for this problem.

## Output
Print the table of 5 in the format as shown in the example.

## Examples

### Example 1
**Input:**
```text
There is no input.
```

**Output:**
```text
5 * 1 = 5
5 * 2 = 10
5 * 3 = 15
5 * 4 = 20
5 * 5 = 25
5 * 6 = 30
5 * 7 = 35
5 * 8 = 40
5 * 9 = 45
5 * 10 = 50
```

---

## My Solution Notes
* **Time Complexity:** $\mathcal{O}(1)$ — A loop executing exactly 10 iterations runs in constant time.
* **Space Complexity:** $\mathcal{O}(1)$ — Only a single loop counter variable is allocated in memory.
