# D. Print Z

| Attribute | Specification |
| :--- | :--- |
| **Time Limit per Test** | 1 second |
| **Memory Limit per Test** | 256 megabytes |

## Problem Description
Your task is to print the letter `Z` using asterisks (`*`) as shown below.

## Input
There is no input for this problem.

## Output
Print the following pattern exactly as shown:
```text
*****
   *
  *
 *
*****
```

## Examples

### Example 1
**Input:**
```text
There is no input.
```

**Output:**
```text
*****
   *
  *
 *
*****
```

---

## My Solution Notes
* **Time Complexity:** $\mathcal{O}(1)$ — Printing a static $5 \times 5$ pattern executes in a fixed number of operations.
* **Space Complexity:** $\mathcal{O}(1)$ — No variables or memory arrays are scaled dynamically.
